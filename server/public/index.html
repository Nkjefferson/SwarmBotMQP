Page Tutorial

This website is used to access server data formed by the node robots. There are a number of ways to access the data, which all come from using the tabs on the left side.

The Home Tab

The home tab contains some basic graphs of the data stored on the server. This is a basic view of the data, which can be

The Node Tab

The node tab allows users to directly access particular pieces of data. Using the selector on the node tab, you can pick a particular node to view. Once in that node you can use the tree to find data from individual dates and times.

The Maps Tab

The maps tab shows the location of th driver and node robots on a localized map of the farm. Using settings in the map, you can show heatmaps of the most recent data collected by each node.
